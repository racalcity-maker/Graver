# Tools

Здесь будут вспомогательные утилиты для разработки и производства.

Планируемые скрипты:
- упаковка frontend build в assets для firmware;
- локальная валидация `manifest.json`;
- генерация тестовых raster jobs;
- CRC/size check для upload;
- экспорт machine config для конкретной платы.

Утилиты лучше держать вне firmware, чтобы не смешивать build-пайплайн с кодом устройства.
